# VitalityFlow - Página de Vendas

Página de vendas otimizada para VitalityFlow, focada em performance e conversão.

## Estrutura do Projeto

```
VitalityFlow-PaginaVendas/
├── pagina/
│   ├── code.html          # Página de vendas principal (otimizada)
│   ├── screen.png         # Screenshot da página
│   └── server.js          # Servidor Node.js simples
├── pagina.html            # Cópia da página (backup)
├── servir-pagina.js       # Script para servir a página localmente
├── iniciar-servidor.bat   # Script para iniciar servidor no Windows
└── README.md              # Este arquivo

```

## Como Usar

### Opção 1: Servidor Node.js Simples

Execute o servidor Node.js:

```bash
node servir-pagina.js
```

A página estará disponível em: `http://localhost:8080`

### Opção 2: Script Batch (Windows)

Clique duas vezes em `iniciar-servidor.bat` ou execute:

```bash
.\iniciar-servidor.bat
```

### Opção 3: Servidor na pasta pagina

```bash
cd pagina
node server.js
```
